### Library Capstone Project

## MERN Fullstack JavaScript Web Program
  This Application is built as a full stack (MERN: MongoDB, Express, React, Node.js) web project. JavaScript code is used to seed the Database cluster on MongoDB, to initiate the server using Express with the web on port http://localhost:5000, and to display the webpage on port 5173 using React. The purpose of the project is to create a library of books. The functions the website offers are to check in and check out a book and check status of a book.

### Technologies used:
  JavaScript(Node.js)
  MongoDB
  Express
  React/Vite
  NPM
